# Четвёртое задание для Bellintegrator
&nbsp;&nbsp;&nbsp;&nbsp; Четвёртое вступительное задание для компании Bellintegrator — решение алгоритмической задачи на проверку корректности скобочной последовательности.

## **Особенности:** 
1. Алгоритм работает за O(n) по времени и памяти.
2. Присутствуют юнит-тесты для проверки корректности решения.
